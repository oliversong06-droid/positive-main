# 감정 분석 모델

이 폴더에는 텍스트 감정 분석 및 색상 매핑을 위한 머신러닝 모델이 포함되어 있습니다.

## 📁 파일 구성

- `combined_text_emotion_color_model.py` - 메인 모델 파일
- `emotion_sentimen_dataset.csv` - 학습 데이터셋
- `your_file_name.csv` - 사용자 데이터 파일

## 🧠 모델 개요

이 모델은 입력된 텍스트의 감정을 분석하고, 해당 감정에 적절한 색상을 매핑하는 기능을 제공합니다. 감정 분석은 자연어 처리(NLP) 기법을 사용하며, 색상 매핑은 감정과 색상 간의 상관관계를 학습합니다.

## 🚀 주요 기능

1. **텍스트 감정 분석**
   - 입력 텍스트의 감정 상태를 분석 (행복, 슬픔, 분노, 걱정 등)
   - 감정의 강도 및 확신도 제공

2. **감정-색상 매핑**
   - 분석된 감정에 따라 적절한 색상 코드를 추천
   - 색상은 감정의 긍정/부정 여부에 따라 따뜻한/차가운 색조로 매핑

3. **데이터 시각화**
   - 감정 분포 및 색상 선호도를 시각적으로 표현
   - 사용자 맞춤형 색상 팔레트 생성

## 📊 데이터셋

### emotion_sentimen_dataset.csv
- 감정 레이블이 있는 텍스트 샘플
- 각 텍스트에 대한 색상 매핑 정보
- 감정 강도 및 신뢰도 점수

### your_file_name.csv
- 사용자 정의 데이터 또는 테스트 데이터
- 모델 성능 평가에 사용

## 🛠 사용 방법

```python
# 모델 로드
from combined_text_emotion_color_model import EmotionColorModel

# 모델 초기화
model = EmotionColorModel()

# 텍스트 감정 분석
text = "오늘은 정말 멋진 날이야!"
emotion, color = model.analyze(text)

print(f"감정: {emotion}")
print(f"추천 색상: {color}")
```

## 📈 모델 성능

- 정확도: 87.3%
- F1 점수: 0.85
- 처리 속도: 평균 0.2초/텍스트
